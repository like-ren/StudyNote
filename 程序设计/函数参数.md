# 函数参数

本文介绍了函数参数的多种设置方法

## 一、数组作为函数参数

### 1.指针传递  
` void test(int *arr[],int n) {} `

### 2.数组名  
` void test(int arr[],int n) {} `  
实质上仍然是指针

### 3.引用传递
` void test(int (&arr)[10]) {} `  
必须指明数组的大小

### 4.const引用
` void test(const int (&arr)[10]){} `  

### 5.引用传递（不指定规模）  
` template<size_t N> `  
` void test(int (&arr)[N]) {} `  

### 6.const引用（不指定规模）  
` template<size_t N> `  
` void test(const int(&arr)[N]) {} `
