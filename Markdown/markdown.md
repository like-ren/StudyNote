# MARKDOWN语法指南

## 1.快捷键  
* Ctrl+B  加粗
* Ctrl+I  斜体
* Ctrl+L  插入链接
* Ctrl+K  插入代码
* Ctrl+G  插入图片

## 2.基本语法  
### 2.1字体
使用 * 文字 * 或者 _ 文字 _ 表示
* 单个*/_  倾斜
* 两个*/_  加粗
* 三个*/_  倾斜加粗
* 两个~    删除线

### 2.2标题
使用#进行表示（#后要加空格）
* #一级标题
* ##二级标题
* ###三级标题
* ####四级标题

### 2.3换行
* 两次换行
* 本行两次空格+换行

### 2.4分点
使用+-*或1.2.3.表示（后要加空格）
- 无序
  使用*/+/-表示
- 有序
  数字加.

### 2.5高光
使用反引号表示（键盘左上角的小点）
- /`代码/`
